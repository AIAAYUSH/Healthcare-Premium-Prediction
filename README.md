# 🏥 Healthcare Premium Prediction (Regression)

## 🚀 Live Demo

👉 https://healthcare-premium-prediction-project-new.streamlit.app/

---

## 📌 Project Overview

This project predicts **health insurance premiums** using Machine Learning regression models.
It demonstrates a complete ML pipeline — from **data preprocessing to deployment using Streamlit**.

---

## ✨ Features

* 📊 Exploratory Data Analysis (EDA)
* 🧹 Data Cleaning & Feature Engineering
* 🤖 Regression Models (Linear, Ridge, Lasso, XGBoost)
* 📈 Model Evaluation (R², MAE, RMSE)
* 🌐 Streamlit Web App for real-time predictions

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* Matplotlib, Seaborn
* Streamlit

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/AIAAYUSH/Healthcare-Premium-Prediction.git
cd Healthcare-Premium-Prediction
```

### 2️⃣ Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install Dependencies

```
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```
cd app
streamlit run main.py
```

Then open:
👉 http://localhost:8501
